# Code and Focus

Use code fences

<CH.Code>

```js foo.js
console.log();
```

</CH.Code>

Works by default with many languages.

```groovy
def f() {
   return [2, 3]
}

(a, b) = f()
println a
println b
```

Use `focus` to show the code that's important to the reader

```groovy focus=5
def f() {
   return [2, 3]
}

(a, b) = f()
println a
println b
```

You can specify a list of line numbers

```groovy focus=1:3,6
def f() {
   return [2, 3]
}

(a, b) = f()
println a
```

And also columns

```groovy focus=2,5[1:6]
def f() {
   return [2, 3]
}

(a, b) = f()
println a
```
