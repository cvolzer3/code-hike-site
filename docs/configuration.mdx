Somewhere in your code you'll have a _`remarkPlugins`_ array, there you'll find Code Hike's [config object](focus://9:13). If you haven't already set Code Hike up, see the [installation docs](installation) first.

```js your.config.js
const remarkSomething = require("remark-something")
// focus
const { remarkCodeHike } = require("@code-hike/mdx")
const someTheme = require("shiki/themes/nord.json")

mdxOptions = {
  remarkPlugins: [
    // focus(1:8)

      remarkCodeHike,
      {
        // link[9:26] #line-numbers
        lineNumbers: false,
        // link[9:24] #auto-import
        autoImport: true,
        // link[9:24] #theme
        theme: someTheme,
      },
    ],
    remarkSomething,
    remarkSomethingElse,
  ],
}
```

## Auto Import

<CH.Section>

By default, any Code Hike component used in your mdx files will be automatically imported. But there are some tools that doesn't work well with imports in mdx files, so you can disable this feature by setting _`autoImport: false`_.

```js
mdxOptions = {
  remarkPlugins: [
    [
      remarkCodeHike,
      {
        theme: someTheme,
        // focus
        // mark[21:25]
        autoImport: false,
      },
    ],
  ],
}
```

</CH.Section>

Then you'll need to pass Code Hike components as a prop like this:

```js
import Example from "./example.mdx"
import { CH } from "@code-hike/mdx/components"

function App() {
  // mark[19:37]
  return <Example components={{ CH }} />
}
```
