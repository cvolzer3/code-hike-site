Code Hike is an MDX plug in for rendering code blocks with superpowers.

### Features

Syntax highlighting • code tabs • multiple files • annotations • code links • code focus • interactive components • diff transitions • inline syntax highlighting • previews • and more!

Code Hike is **not** for you if:

- You are not using React
- You don't want to use MDX
- You are not using MDX v2 (or xdm)
