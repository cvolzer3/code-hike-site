There are several ways to write code inside markdown. The most common is to wrap the code using triple backticks:

{/* prettier-ignore */}
````mdx lorem.mdx focus=3:5
## Lorem ipsum

```
print("Hello, world!")
```

dolor **sit** amet
````

### Syntax highlighting

If you specify a language after the <CH.InlineCode>```</CH.InlineCode> Code Hike will apply syntax highlighting to the code.

### Filename

If you add a filename after the language, Code Hike will show a different UI:

````mdx lorem.mdx focus=3:5 mark=3[11:18]
## Lorem ipsum

```python hello.py
print("Hello, world!")
```

dolor **sit** amet
````
